## Installation / Run

---

The following components must be installed locally:

- [nodejs](https://nodejs.org/en/) v13.2.0

To run the project locally, enter the following in the command line / bash:

```console
$ git clone <linktorepository>
$ cd <repositoryname>
$ npm install
$ npm run dev
```
